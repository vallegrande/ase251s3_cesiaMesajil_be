# Implementación del Endpoint “Hola Mundo” – Spring Boot

## Descripción
En esta actividad se desarrolló un servicio backend básico utilizando Spring Boot, el cual implementa un endpoint de tipo GET que muestra el mensaje **“Hola Mundo!”** en el navegador.

---

## Paso a paso de la implementación

1. Se ingresó a Spring Initializr para crear un nuevo proyecto backend utilizando el framework Spring Boot.

2. En la configuración del proyecto se seleccionó el lenguaje **Java**, el gestor de dependencias **Maven** y se agregó la dependencia **Spring Web**, necesaria para la creación de servicios web.

3. Una vez completada la configuración, el proyecto fue generado y descargado en formato comprimido.

4. El archivo descargado fue descomprimido y abierto en el entorno de desarrollo (IDE) para iniciar el trabajo con el código.

5. Dentro del proyecto, se creó una clase controladora donde se utilizó la anotación correspondiente para definir un endpoint de tipo **GET**.

6. El endpoint fue programado para retornar el mensaje **“Hola Mundo!”**, el cual se puede visualizar correctamente al acceder a la URL desde el navegador.

7. Posteriormente, la aplicación fue ejecutada para verificar que el servidor iniciara sin errores y que el endpoint funcionara correctamente.

8. Luego de comprobar el funcionamiento, el código fuente fue subido a un repositorio en GitHub con el nombre **ase251s3_nombreApellido_be**.

9. Finalmente, se elaboró este archivo README describiendo los pasos realizados y se grabó un video con la cámara encendida mostrando todo el proceso de implementación.

---

## ✅ Resultado
El endpoint funciona correctamente y devuelve el mensaje **“Hola Mundo!”** al ser consultado mediante una petición GET.
